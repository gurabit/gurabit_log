Hugoでよく使うコマンド

新しいサイトを作る

hugo new site サイトの名前

Huhoのローカルサーバー立ち上げ

hugo server -w -D

-wはレンダリング　-Dは下書きも表示

サーバーを停止

Ctrl+C

新しい記事を作る

hugo new content/記事のジャンル名/記事の名前.md

公開したい記事のdraftをfalseに書き換える

publicフォルダを作る

hugo

Hugoコマンドを入力するとpublicフォルダが作られる

Hugoのバージョンを確認する

hugo version

現在のHugoのバージョンの値が返ってくる

https://maku77.github.io/hugo/
